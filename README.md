# create-vtems

快速创建 vue 项目模板

```
E:\tems> vtems
✔ 请输入项目名称： vtems_project
✔ 是否使用 TypeScript 语法？ 是
✔ 是否引入 Vue Router 进行单页面应用开发？ 是
✔ 是否启用 axios 接口请求？ 是
✔ 是否启用 ui？ element-plus
✔ ui是否按需加载？ 是
✔ 是否使用css预处理器？ 是
正在初始化项目 E:\tems\vtems_project
项目初始化完成，可执行以下命令：
cd vtems_project
pnpm install
pnpm dev
```

## 用法

在终端中运行以下命令:

```sh
pnpm create vtems
```
